# ⚡ Digital Logic: Simulation to Hardware
This project demonstrates the complete engineering workflow: designing a logic system in a simulator and building the physical prototype.

### 🔍 Project Overview
The uploaded photo shows the **NI MultiSim schematic** acting as the blueprint for the **Physical Breadboard** construction. 

* **Software:** NI MultiSim (used for logic verification).
* **Hardware:** Breadboard, Logic ICs, and LEDs.

### 🛠️ Verification Process
By having the simulation and the physical build in one view, I verified that the hardware correctly matches the theoretical logic. This "side-by-side" approach ensures that the physical wiring follows the designed schematic perfectly.
